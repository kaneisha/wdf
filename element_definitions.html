Element Definitions

h1
The most important heading of the page goes under the h1 which means header 1.
<h1>Example of header 1</h1>
h2
The second most important heading goes under the h2 which means header 2(first subheading)
<h2>Example of header 2</h2>
h3
The third most important content of the page this is like a sub heading to the first subheading, this means header 3
<h3>Example of header 3</h3>
p
This stands for paragraph. This is where you'll put your content.<p>Example of paragraph
</p>
ul
This stands for unordered list. You use this when you want to make a bulleted list or anything for any order.
<ul>
<li>This is a list item</li>
</ul>
ol
This stands for ordered list. You use this when you want to make a numbered list or anything that's a specific order.
<ol>
<li>This is a list item</li>
</ol>
li
This stands for list item. You use in between a ul or ol to show the list.
<ul>
<li>This is a list item</li>
</ul>
